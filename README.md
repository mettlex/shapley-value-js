A TypeScript/JavaScript library for calculating Shapley Value

```sh
npm install shapley
```

## Usage

```js
const shapleyValueForP = shapley(N, v)(p)
// N = set of players
// v = value function
// p = current player
```
For examples, please check [examples](./examples/) folder.
